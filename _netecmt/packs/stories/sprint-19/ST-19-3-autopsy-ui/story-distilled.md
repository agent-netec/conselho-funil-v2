# Story Distilled: ST-19-3-autopsy-ui

## Contexto
Interface de usuário para o diagnóstico Autopsy. Deve transmitir a sensação de um "laboratório forense" ou "necropsia estratégica".

## Requisitos Técnicos
- Utilizar componentes Radix UI e Tailwind CSS 4.
- Implementar animações de carregamento (Framer Motion).
- Exibir recomendações em cards categorizados por prioridade.

## Definição de Pronto (DoD)
- [ ] Interface reflete fielmente os dados do relatório Autopsy.
- [ ] Responsividade garantida.
- [ ] Transições suaves entre estados de carregamento e resultado.
