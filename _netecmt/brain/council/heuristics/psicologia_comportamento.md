# Heurísticas: Psicologia & Comportamento

## Contexto de Aplicação
Regras práticas para trabalhar com a natureza humana em marketing, não contra ela. Aplicável a qualquer funil que busca ação/compra.

## 16 Regras de Psicologia & Comportamento

### 1. Se ignoras o comportamento real, ENTÃO tua estratégia falha
**Porque**: Pessoas fazem, não apenas pensam; comportamento ≠ intenção.
**Exceção**: Nenhuma - sempre observar comportamento real.
**Aplicação**: Testar e medir; não assumir.

### 2. Se há muita fricção, ENTÃO conversão cai
**Porque**: Cada barreira remove uma % de clientes.
**Exceção**: Nenhuma - sempre minimizar fricção.
**Aplicação**: Remover campos de formulário, simplificar checkout, reduzir cliques.

### 3. Se o timing está errado, ENTÃO mensagem não funciona
**Porque**: Contexto é tudo; mensagem certa, hora errada = falha.
**Exceção**: Nenhuma - sempre considerar timing.
**Aplicação**: Enviar email no momento certo (baseado em comportamento).

### 4. Se não há automação, ENTÃO não escala
**Porque**: Manual não funciona em escala.
**Exceção**: Nenhuma - sempre automatizar.
**Aplicação**: Criar sequências automáticas baseadas em comportamento.

### 5. Se não admites fraquezas, ENTÃO cliente não confia
**Porque**: Perfeição parece falsa; honestidade cria confiança.
**Exceção**: Nenhuma - sempre ser honesto sobre limitações.
**Aplicação**: "Este produto não é para você se...", "Aqui está o que não funciona...".

### 6. Se não há dados, ENTÃO estás apenas adivinhando
**Porque**: Intuição falha; dados não mentem.
**Exceção**: Nenhuma - sempre medir.
**Aplicação**: Rastrear cliques, conversões, comportamento, tempo gasto.

### 7. Se o funil é muito complexo, ENTÃO conversão cai
**Porque**: Simplicidade funciona; complexidade confunde.
**Exceção**: Nenhuma - sempre simplificar.
**Aplicação**: Menos etapas = mais conversão.

### 8. Se não há sequência, ENTÃO mensagens isoladas falham
**Porque**: Sequência cria contexto; contexto cria compreensão.
**Exceção**: Nenhuma - sempre ter sequência.
**Aplicação**: Email 1 → Email 2 → Email 3... (cada um construindo sobre anterior).

### 9. Se ignoras sinais de alerta, ENTÃO clientes ruins entram
**Porque**: Comportamento inicial prediz comportamento futuro.
**Exceção**: Nenhuma - sempre observar sinais.
**Aplicação**: Se cliente não responde, não engaja, provavelmente não vai comprar.

### 10. Se não há padrão, ENTÃO cliente fica confuso
**Porque**: Padrão cria previsibilidade; previsibilidade cria conforto.
**Exceção**: Nenhuma - sempre ter padrão consistente.
**Aplicação**: Mesmo tom, mesmo estilo, mesma frequência em comunicações.

### 11. Se a promessa não é credível, ENTÃO cliente não acredita
**Porque**: Promessas exageradas parecem falsas.
**Exceção**: Nenhuma - sempre ser conservador em promessas.
**Aplicação**: Prometer menos, entregar mais.

### 12. Se não há "por que agora", ENTÃO cliente adia
**Porque**: Sem urgência, "depois" nunca chega.
**Exceção**: Nenhuma - sempre criar razão para agir agora.
**Aplicação**: Deadline, escassez, problema que precisa ser resolvido hoje.

### 13. Se o cliente não se vê na mensagem, ENTÃO não se identifica
**Porque**: Identificação cria conexão; conexão cria ação.
**Exceção**: Nenhuma - sempre falar com cliente específico.
**Aplicação**: Usar linguagem, problemas, aspirações do cliente.

### 14. Se não há validação social, ENTÃO cliente duvida
**Porque**: Pessoas seguem o rebanho; prova social vende.
**Exceção**: Nenhuma - sempre incluir prova social.
**Aplicação**: Testimoniais, números ("1000+ clientes"), reviews.

### 15. Se o funil não testa hipóteses, ENTÃO não melhora
**Porque**: Sem teste, não há aprendizado.
**Exceção**: Nenhuma - sempre testar.
**Aplicação**: A/B testar headlines, copy, imagens, CTAs, timing.

### 16. Se ignoras a jornada emocional, ENTÃO copy é racional demais
**Porque**: Emoção move; razão justifica.
**Exceção**: Nenhuma - sempre incluir emoção.
**Aplicação**: Contar histórias, criar conexão emocional, não apenas listar benefícios.

## Checklist Rápido de Psicologia

- [ ] Observei comportamento real (não suposições)?
- [ ] Minimizei fricção?
- [ ] Considerei timing correto?
- [ ] Automatizei sequências?
- [ ] Admiti fraquezas/limitações?
- [ ] Tenho dados para suportar decisões?
- [ ] Simplifiquei o máximo possível?
- [ ] Tenho sequência clara?
- [ ] Observei sinais de alerta?
- [ ] Há padrão consistente?
- [ ] Promessa é credível?
- [ ] Há urgência/razão para agir agora?
- [ ] Cliente se vê na mensagem?
- [ ] Há prova social?
- [ ] Testei hipóteses?
- [ ] Incluí emoção?

## Princípios de Behavioral Dynamics

| Princípio | Aplicação | Exemplo |
|-----------|-----------|---------|
| **Fricção** | Remover barreiras | Checkout em 1 clique vs 5 passos |
| **Timing** | Mensagem no momento certo | Email quando cliente está engajado |
| **Sequência** | Contexto progressivo | Email 1 (problema) → Email 2 (solução) |
| **Automação** | Sistemas que funcionam sem você | Trigger: Clique → Ação: Email automático |
| **Honestidade** | Admitir limitações | "Não funciona para X" cria confiança |
| **Dados** | Medir tudo | Rastrear cliques, tempo, comportamento |
| **Simplicidade** | Menos é mais | 1 oferta > 5 ofertas |
| **Validação Social** | Prova que funciona | Testimoniais, números, reviews |
| **Emoção** | Conectar, não apenas informar | Histórias vs listas de benefícios |
| **Padrão** | Consistência cria conforto | Mesmo tom, frequência, estilo |

## Métricas de Comportamento

| Métrica | O que Significa | Ação |
|---------|-----------------|------|
| **Bounce Rate Alto** | Muita fricção ou mensagem errada | Simplificar, testar copy |
| **Tempo Baixo na Página** | Não engajado | Melhorar headline, adicionar elemento visual |
| **Taxa de Clique Baixa** | CTA não claro ou não atrativo | Testar CTAs diferentes |
| **Abandono de Carrinho** | Fricção no checkout | Simplificar processo |
| **Não-abertura de Email** | Assunto não atrativo | Testar assuntos |
| **Não-clique em Email** | Conteúdo não relevante | Segmentar lista, personalizar |
| **Não-resposta** | Cliente não engajado | Remover da sequência |
| **Comportamento Errático** | Cliente confuso | Simplificar funil |
