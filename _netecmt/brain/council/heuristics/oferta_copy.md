# Heurísticas: Oferta & Copy

## Contexto de Aplicação
Regras práticas para criar ofertas irresistíveis e copywriting que converte. Aplicável a qualquer comunicação de marketing que busca ação/venda.

## 18 Regras de Oferta & Copy

### 1. Se não há oferta clara, ENTÃO não há ação esperada
**Porque**: Sem oferta, cliente não sabe o que fazer.
**Exceção**: Conteúdo puramente informativo (mas mesmo assim, melhor ter CTA).
**Aplicação**: Toda comunicação deve ter oferta explícita.

### 2. Se a promessa não é óbvia, ENTÃO o cliente não vai entender o benefício
**Porque**: Cliente não lê; ele escaneia.
**Exceção**: Nenhuma - promessa deve ser cristalina.
**Aplicação**: Promessa deve estar em headline, primeira linha e resumo.

### 3. Se não há prova, ENTÃO cliente não acredita
**Porque**: Afirmações sem evidência não vendem.
**Exceção**: Marca com credibilidade estabelecida.
**Aplicação**: Incluir testimoniais, dados, estudos de caso.

### 4. Se a headline não chama atenção, ENTÃO ninguém lê o resto
**Porque**: 80% das pessoas leem só o headline.
**Exceção**: Nenhuma - headline é crítica.
**Aplicação**: Testar múltiplos headlines; escolher o que mais chama atenção.

### 5. Se o copy é sobre você, ENTÃO cliente não se importa
**Porque**: Cliente quer saber "e daí?" para ele.
**Exceção**: Nenhuma - sempre focar no cliente.
**Aplicação**: Reescrever copy focando em benefício do cliente, não em você.

### 6. Se não há urgência, ENTÃO cliente adia a decisão
**Porque**: Sem urgência, "depois" nunca chega.
**Exceção**: Nenhuma - sempre criar urgência (real ou percebida).
**Aplicação**: Deadline, oferta limitada, escassez.

### 7. Se o preço não é justificado, ENTÃO cliente acha caro
**Porque**: Preço sem contexto parece alto.
**Exceção**: Nenhuma - sempre justificar preço.
**Aplicação**: Mostrar valor: "Você vai receber X, Y, Z que valem $XXX".

### 8. Se há muitas opções, ENTÃO cliente não escolhe
**Porque**: Paradoxo da escolha paralisa.
**Exceção**: Nenhuma - simplificar opções.
**Aplicação**: Oferecer 1-3 opções, não 10.

### 9. Se o copy é muito longo, ENTÃO cliente não lê
**Porque**: Atenção é limitada.
**Exceção**: Quando copy longo está bem estruturado e envolvente.
**Aplicação**: Usar subheadings, parágrafos curtos, espaço em branco.

### 10. Se não há call-to-action claro, ENTÃO cliente não sabe o que fazer
**Porque**: Próximo passo precisa ser óbvio.
**Exceção**: Nenhuma - sempre ter CTA claro.
**Aplicação**: "Clique aqui", "Compre agora", "Reserve seu lugar".

### 11. Se a oferta é genérica, ENTÃO não diferencia de concorrentes
**Porque**: Genérico não vende; específico vende.
**Exceção**: Nenhuma - sempre ser específico.
**Aplicação**: "Perda de peso" → "Perder 10kg em 30 dias sem dieta".

### 12. Se não há risco reverso, ENTÃO cliente tem medo de comprar
**Porque**: Garantia remove risco.
**Exceção**: Nenhuma - sempre oferecer garantia.
**Aplicação**: "30 dias de garantia dinheiro de volta, sem perguntas".

### 13. Se o copy não tem estrutura, ENTÃO é difícil de ler
**Porque**: Estrutura guia o leitor.
**Exceção**: Nenhuma - sempre ter estrutura clara.
**Aplicação**: Headline → Problema → Solução → Prova → Oferta → CTA.

### 14. Se não há escassez, ENTÃO cliente não se move rápido
**Porque**: Abundância não cria urgência.
**Exceção**: Nenhuma - sempre criar escassez (real ou percebida).
**Aplicação**: "Apenas 5 vagas", "Oferta válida até sexta".

### 15. Se o copy não fala com a linguagem do cliente, ENTÃO ele não se identifica
**Porque**: Linguagem cria conexão.
**Exceção**: Nenhuma - sempre falar a linguagem do cliente.
**Aplicação**: Pesquisar como cliente fala; usar suas palavras.

### 16. Se não há transição suave entre problema e solução, ENTÃO copy é confuso
**Porque**: Transição guia o raciocínio.
**Exceção**: Nenhuma - sempre ter transição clara.
**Aplicação**: "Mas há uma solução...", "Aqui está o segredo...".

### 17. Se o copy não tem storytelling, ENTÃO é chato
**Porque**: Histórias vendem; fatos não.
**Exceção**: Nenhuma - sempre incluir histórias.
**Aplicação**: Contar história de cliente, transformação, resultado.

### 18. Se não há follow-up, ENTÃO deixa dinheiro na mesa
**Porque**: Primeira mensagem não converte todos.
**Exceção**: Nenhuma - sempre ter sequência de follow-up.
**Aplicação**: Email 1, 2, 3... até venda ou opt-out.

## Checklist Rápido de Copy

- [ ] Oferta é clara?
- [ ] Promessa é óbvia?
- [ ] Há prova (testimoniais, dados)?
- [ ] Headline chama atenção?
- [ ] Copy foca no cliente, não em mim?
- [ ] Há urgência?
- [ ] Preço é justificado?
- [ ] Opções são simples (1-3)?
- [ ] Copy é bem estruturado?
- [ ] CTA é claro?
- [ ] Oferta é específica?
- [ ] Há garantia/risco reverso?
- [ ] Estrutura é clara (Headline → Problema → Solução → Prova → Oferta → CTA)?
- [ ] Há escassez?
- [ ] Copy fala linguagem do cliente?
- [ ] Transições são suaves?
- [ ] Há storytelling?
- [ ] Há follow-up planejado?

## Estrutura Clássica de Sales Letter

1. **Headline**: Promessa maior, benefício principal
2. **Abertura**: Problema que cliente enfrenta
3. **Agitação**: Aprofundar o problema, criar urgência
4. **Transição**: "Mas há uma solução..."
5. **Solução**: O que você oferece
6. **Prova**: Testimoniais, dados, estudos de caso
7. **Oferta**: Exatamente o que cliente recebe
8. **Preço**: Justificado pelo valor
9. **Garantia**: Risco reverso
10. **Urgência**: Deadline, escassez
11. **CTA**: Próximo passo claro
12. **PS**: Reforçar oferta ou urgência
