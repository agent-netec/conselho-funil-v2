# Heurísticas: LTV & Retenção

## Contexto de Aplicação
Regras práticas para maximizar o valor de vida do cliente e reter clientes por mais tempo. Aplicável a qualquer negócio com receita recorrente.

## 16 Regras de LTV & Retenção

### 1. Se não segmentas clientes, ENTÃO deixas dinheiro na mesa
**Porque**: Diferentes clientes precisam de diferentes estratégias.
**Exceção**: Nenhuma - sempre segmentar.
**Aplicação**: Segmentar por: valor, comportamento, etapa de vida.

### 2. Se não há estratégia de lifecycle, ENTÃO clientes caem
**Porque**: Diferentes etapas precisam de diferentes mensagens.
**Exceção**: Nenhuma - sempre ter lifecycle strategy.
**Aplicação**: Onboarding → Engajamento → Retenção → Upsell.

### 3. Se não automatizas sequências de retenção, ENTÃO não escala
**Porque**: Manual não funciona em escala.
**Exceção**: Nenhuma - sempre automatizar.
**Aplicação**: Email sequences, SMS, in-app messages automáticas.

### 4. Se não há onboarding, ENTÃO cliente não realiza valor
**Porque**: Sem onboarding, cliente não sabe como usar.
**Exceção**: Nenhuma - sempre ter onboarding.
**Aplicação**: Primeiros 7-30 dias críticos.

### 5. Se não há engajamento contínuo, ENTÃO cliente dorme
**Porque**: Engajamento mantém cliente ativo.
**Exceção**: Nenhuma - sempre engajar.
**Aplicação**: Email, SMS, conteúdo, comunidade.

### 6. Se não há upsell/cross-sell, ENTÃO LTV é baixo
**Porque**: Upsell aumenta valor por cliente.
**Exceção**: Nenhuma - sempre ter oportunidade.
**Aplicação**: Oferecer produtos/serviços complementares.

### 7. Se não há win-back campaign, ENTÃO clientes perdidos não voltam
**Porque**: Win-back é mais barato que aquisição.
**Exceção**: Nenhuma - sempre tentar win-back.
**Aplicação**: Email para clientes inativos.

### 8. Se não há feedback loop, ENTÃO não melhoras
**Porque**: Feedback guia melhoria.
**Exceção**: Nenhuma - sempre coletar feedback.
**Aplicação**: Surveys, NPS, entrevistas.

### 9. Se não há comunidade, ENTÃO cliente se sente sozinho
**Porque**: Comunidade cria conexão e retenção.
**Exceção**: Nenhuma - sempre criar comunidade.
**Aplicação**: Grupo, fórum, eventos, masterminds.

### 10. Se não há suporte excelente, ENTÃO cliente sai
**Porque**: Suporte ruim = churn.
**Exceção**: Nenhuma - sempre ter suporte excelente.
**Aplicação**: Resposta rápida, solução eficaz.

### 11. Se não há dados de retenção, ENTÃO não sabes o que funciona
**Porque**: Sem dados, não há aprendizado.
**Exceção**: Nenhuma - sempre medir retenção.
**Aplicação**: Rastrear churn, LTV, segmentação.

### 12. Se não há segmentação de email, ENTÃO mensagem é genérica
**Porque**: Segmentação = relevância = retenção.
**Exceção**: Nenhuma - sempre segmentar.
**Aplicação**: Email diferente para cada segmento.

### 13. Se não há conteúdo de valor, ENTÃO cliente não vê ROI
**Porque**: Valor mantém cliente.
**Exceção**: Nenhuma - sempre entregar valor.
**Aplicação**: Blog, vídeos, webinars, templates.

### 14. Se não há reconhecimento de cliente, ENTÃO se sente invisível
**Porque**: Reconhecimento cria lealdade.
**Exceção**: Nenhuma - sempre reconhecer.
**Aplicação**: Aniversários, milestones, VIP treatment.

### 15. Se não há preço de retenção, ENTÃO cliente sai por preço
**Porque**: Retenção é mais barato que aquisição.
**Exceção**: Nenhuma - sempre oferecer preço de retenção.
**Aplicação**: Desconto para renovação, loyalty program.

### 16. Se não há análise de churn, ENTÃO não sabes por que saem
**Porque**: Sem análise, não há melhoria.
**Exceção**: Nenhuma - sempre analisar churn.
**Aplicação**: Exit surveys, análise de dados.

## Etapas do Lifecycle

| Etapa | Objetivo | Ações | Duração |
|-------|----------|-------|---------|
| **Onboarding** | Realizar valor | Welcome email, tutorial, setup | 7-30 dias |
| **Engajamento** | Manter ativo | Conteúdo, dicas, comunidade | 30-90 dias |
| **Retenção** | Manter cliente | Email, suporte, reconhecimento | Contínuo |
| **Upsell** | Aumentar valor | Oferecer produtos maiores | Quando pronto |
| **Advocacy** | Fazer referências | Referral program, testimonial | Quando satisfeito |
| **Win-back** | Trazer de volta | Email especial, oferta | Quando inativo |

## Segmentação de Clientes

| Segmento | Características | Estratégia |
|----------|-----------------|-----------|
| **VIP** | Alto LTV, engajado | Atendimento premium, acesso exclusivo |
| **Core** | LTV médio, engajado | Conteúdo regular, suporte bom |
| **At-Risk** | Engajamento baixo | Re-engajamento, oferta especial |
| **Churned** | Saiu | Win-back, oferta de retorno |
| **Novo** | Recém-chegado | Onboarding intensivo |

## Métricas de Retenção

| Métrica | Ruim | OK | Bom | Excelente |
|---------|------|-----|-----|-----------|
| Taxa de Retenção Mensal | <80% | 80-90% | 90-95% | >95% |
| Taxa de Churn Mensal | >20% | 10-20% | 5-10% | <5% |
| LTV (vs CAC) | <3x | 3-5x | 5-10x | >10x |
| Tempo Médio de Cliente | <6 meses | 6-12 meses | 1-2 anos | >2 anos |
| NPS (Net Promoter Score) | <30 | 30-50 | 50-70 | >70 |
| Taxa de Upsell | <10% | 10-20% | 20-40% | >40% |
| Taxa de Referência | <5% | 5-10% | 10-20% | >20% |
| Satisfação (CSAT) | <70% | 70-80% | 80-90% | >90% |

## Fórmula de LTV

**LTV = (Valor Médio de Pedido) × (Frequência de Compra) × (Tempo Médio de Cliente)**

Exemplo: $100 × 12 × 3 anos = $3,600 LTV
