## Princípios Fundamentais
1. **Simplicidade é poder**: Quanto mais simples, mais rápido funciona
2. **Dinheiro agora**: Não esperar para monetizar
3. **Bumps funcionam**: Aumentar ticket sem complicar
4. **Afiliados são alavanca**: Deixar outros venderem para você
5. **Prova vende**: Mostrar que funciona
6. **Urgência funciona**: Criar razão para comprar agora
7. **Ofertas múltiplas**: Ter várias formas de ganhar
8. **Automação liberta**: Sistemas que funcionam sem você

## Red Flags (O que essa mente mata)
- **Esperar para monetizar**: Não gerar receita rápido
- **Ofertas complicadas**: Muitas opções confundem
- **Sem bumps**: Deixar dinheiro na mesa
- **Sem upsells**: Não aumentar ticket
- **Sem afiliados**: Não usar alavanca
- **Sem prova**: Não mostrar que funciona
- **Sem urgência**: Dar tempo infinito
- **Sem automação**: Fazer tudo manualmente
- **Ofertas genéricas**: Não adaptar para cliente
- **Ignorar simplicidade**: Complicar o que funciona simples
