# Playbook: Criação de Conteúdo Viral

## Objetivo

Criar conteúdo com alta probabilidade de viralização em social media.

## Duração

2-4 horas para criar e publicar

## Pré-requisitos

- Entender seu público
- Conhecer trending audio/topics
- Acesso a ferramentas de edição
- Plataforma escolhida

## Etapas do Playbook

### Etapa 1: Escolher Formato (15 min)

1. **Qual plataforma?** (TikTok, Instagram, X, LinkedIn)
2. **Qual formato?** (Reel, Post, Thread, Story)
3. **Qual comprimento?** (15-60s, 15-90s, 5-15 tweets, etc)

### Etapa 2: Definir Hook (30 min)

1. **Qual emoção?** (Curiosidade, Raiva, Alegria, Medo, Inspiração)
2. **Qual hook?** (Pergunta, Provocação, Surpresa, Movimento)
3. **Testar hook** (Você clicaria?)

### Etapa 3: Estruturar Conteúdo (30 min)

1. **Problema** (0-5s): Qual é o problema?
2. **Solução** (5-15s): Qual é a solução?
3. **Cliffhanger** (15-20s): Por que continuar?
4. **CTA** (20-60s): O que fazer?

### Etapa 4: Escolher Trending Audio (10 min)

1. **Qual plataforma?** (TikTok Discover, Instagram Explore)
2. **Qual trending audio?** (Escolha que combina com conteúdo)
3. **Verificar trending** (Está ainda em alta?)

### Etapa 5: Editar Conteúdo (60 min)

1. **Gravar/Editar** (Seguir estrutura)
2. **Adicionar texto on-screen** (Resumir em 1-2 linhas)
3. **Adicionar trending audio** (Sincronizar com conteúdo)
4. **Revisar** (Prende atenção? Ritmo rápido?)

### Etapa 6: Otimizar para Plataforma (15 min)

1. **Captions** (Relevantes, com CTA)
2. **Hashtags** (15-30, relevantes)
3. **Link** (Bio atualizada, link pronto)
4. **CTA** (Claro e específico)

### Etapa 7: Publicar e Monitorar (15 min)

1. **Publicar** (Horário otimizado)
2. **Monitorar** (Primeiras 2 horas)
3. **Responder comentários** (Engajamento)
4. **Rastrear métricas** (Watch time, engajamento)

### Etapa 8: Iterar (30 min)

1. **Analisar** (O que funcionou?)
2. **Identificar** (Hook? Ritmo? Conteúdo?)
3. **Melhorar** (Próximo vídeo)

## Checklist de Publicação

- [ ] Hook nos primeiros 0.5-1s?
- [ ] Estrutura clara (Problema → Solução → CTA)?
- [ ] Trending audio?
- [ ] Texto on-screen?
- [ ] Captions com CTA?
- [ ] Hashtags relevantes?
- [ ] Link na bio?
- [ ] Publicado em horário certo?
- [ ] Respondendo comentários?
- [ ] Rastreando métricas?

## Hooks Que Funcionam

**Curiosidade**: "Você não vai acreditar no que descobri"
**Provocação**: "Você está fazendo isso errado"
**Surpresa**: "Espera até o final"
**Medo**: "Se você não souber isso, vai..."
**Inspiração**: "Como eu consegui..."

## Estrutura por Plataforma

### TikTok
Hook (0-0.5s) → Problema (0.5-5s) → Solução (5-15s) → CTA (15-60s)

### Instagram
Hook (0-1s) → Problema (1-5s) → Solução (5-20s) → CTA (20-90s)

### X/Twitter
Hook (Tweet 1) → Contexto (Tweets 2-5) → Conteúdo (Tweets 6-12) → CTA (Tweets 13-15)

### LinkedIn
Hook (Parágrafo 1) → Contexto (Parágrafos 2-4) → Conteúdo (Parágrafos 5-8) → CTA (Parágrafo 9)

## Métricas Alvo

| Métrica | Alvo |
|---------|------|
| **Watch Time** | >70% |
| **Engajamento** | >3% |
| **Compartilhamentos** | >1% |
| **Seguidores Ganhos** | >50 |
