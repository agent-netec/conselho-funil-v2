# Heuristics: X/Twitter

## Contexto de Aplicação

Regras práticas para criar conteúdo que funciona em X/Twitter. X prioriza **viralidade** (retweets, quotes, replies) e **engajamento rápido**.

## 14 Regras de X/Twitter

### 1. Se primeiro tweet não é atrativo, ENTÃO thread morre
**Porque**: X favorece threads que começam forte.
**Exceção**: Nenhuma.
**Aplicação**: Primeiro tweet deve ser curiosidade/provocação.

### 2. Se thread é muito longa, ENTÃO menos engajamento
**Porque**: X favorece threads entre 5-15 tweets.
**Exceção**: Nenhuma.
**Aplicação**: Mantenha entre 5-15 tweets.

### 3. Se não há engajamento esperado, ENTÃO algoritmo não promove
**Porque**: X prioriza engajamento (retweets, quotes).
**Exceção**: Nenhuma.
**Aplicação**: Crie conteúdo que gera retweets.

### 4. Se tweet não tem CTA, ENTÃO menos engajamento
**Porque**: Sem CTA, menos pessoas engajam.
**Exceção**: Nenhuma.
**Aplicação**: "Retweet se concorda", "Qual é sua opinião?".

### 5. Se não há provocação/opinião, ENTÃO menos viralidade
**Porque**: X favorece conteúdo que gera debate.
**Exceção**: Nenhuma.
**Aplicação**: Tenha opinião forte, não seja neutro.

### 6. Se tweet é muito longo, ENTÃO menos legibilidade
**Porque**: X favorece tweets concisos.
**Exceção**: Nenhuma.
**Aplicação**: Máximo 280 caracteres, seja conciso.

### 7. Se não há link/imagem, ENTÃO menos cliques
**Porque**: Links e imagens aumentam cliques.
**Exceção**: Nenhuma.
**Aplicação**: Sempre inclua link ou imagem.

### 8. Se publica em horário errado, ENTÃO menos impressões
**Porque**: Horário afeta alcance.
**Exceção**: Nenhuma.
**Aplicação**: Melhor: 8-10am, 12-1pm, 5-7pm.

### 9. Se não há consistência, ENTÃO crescimento lento
**Porque**: X favorece criadores consistentes.
**Exceção**: Nenhuma.
**Aplicação**: Publique 5-10 tweets por dia.

### 10. Se não há engajamento com comunidade, ENTÃO menos crescimento
**Porque**: X favorece quem engaja com outros.
**Exceção**: Nenhuma.
**Aplicação**: Responda tweets, comente, retweet.

### 11. Se não há quote tweets, ENTÃO menos viralidade
**Porque**: Quote tweets amplificam alcance.
**Exceção**: Nenhuma.
**Aplicação**: Encoraje quote tweets com conteúdo provocador.

### 12. Se thread não tem numeração, ENTÃO menos legibilidade
**Porque**: Numeração ajuda legibilidade.
**Exceção**: Nenhuma.
**Aplicação**: "1/15", "2/15", etc.

### 13. Se não há call-to-action para seguir, ENTÃO menos seguidores
**Porque**: Sem CTA, menos pessoas seguem.
**Exceção**: Nenhuma.
**Aplicação**: "Siga para mais threads", "Siga para insights".

### 14. Se não há retargeting com Ads, ENTÃO deixas dinheiro na mesa
**Porque**: Ads amplificam alcance.
**Exceção**: Nenhuma.
**Aplicação**: Retargete com anúncios.

## Checklist Rápido de X/Twitter

- [ ] Primeiro tweet é atrativo?
- [ ] Thread entre 5-15 tweets?
- [ ] Engajamento esperado?
- [ ] CTA claro?
- [ ] Há provocação/opinião?
- [ ] Tweet é conciso?
- [ ] Há link/imagem?
- [ ] Publicado em horário certo?
- [ ] Consistência (5-10/dia)?
- [ ] Engajamento com comunidade?
- [ ] Encoraja quote tweets?
- [ ] Thread tem numeração?
- [ ] CTA para seguir?
- [ ] Retargeting planejado?

## Estrutura de Thread X/Twitter

**Tweet 1**: Hook (curiosidade, provocação)
**Tweet 2-3**: Contexto (por que isso importa)
**Tweet 4-10**: Conteúdo (insights, dados, história)
**Tweet 11-14**: Conclusão (takeaway, CTA)
**Tweet 15**: CTA para seguir

## Tipos de Conteúdo que Funcionam

- **Threads de insights** (5-15 tweets com aprendizado)
- **Hot takes** (opinião provocadora)
- **Dados/estatísticas** (com contexto)
- **Histórias pessoais** (vulnerabilidade + lição)
- **Perguntas** (engajamento direto)

## Métricas de Referência

| Métrica | Ruim | OK | Bom | Excelente |
|---------|------|-----|-----|-----------|
| **Retweets/Impressions** | <0.5% | 0.5-1% | 1-3% | >3% |
| **Quotes** | <0.2% | 0.2-0.5% | 0.5-1% | >1% |
| **Replies** | <0.3% | 0.3-0.7% | 0.7-2% | >2% |
| **Likes** | <1% | 1-3% | 3-7% | >7% |
| **Followers Gained** | <5 | 5-20 | 20-100 | >100 |
