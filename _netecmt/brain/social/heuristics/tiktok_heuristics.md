# Heuristics: TikTok

## Contexto de Aplicação

Regras práticas para criar conteúdo que funciona em TikTok. TikTok prioriza **retenção** (watch time) acima de tudo.

## 16 Regras de TikTok

### 1. Se não há hook nos primeiros 0.5 segundos, ENTÃO vídeo morre
**Porque**: TikTok mede retenção. Se pessoa sai nos primeiros 0.5s, algoritmo não promove.
**Exceção**: Nenhuma.
**Aplicação**: Hook visual ou verbal que desperte curiosidade/emoção.

### 2. Se vídeo é mais de 60 segundos, ENTÃO menos alcance
**Porque**: TikTok favorece vídeos que as pessoas assistem completamente.
**Exceção**: Nenhuma (a menos que seja conteúdo muito engajador).
**Aplicação**: Mantenha entre 15-60 segundos.

### 3. Se ritmo é lento, ENTÃO pessoas saem
**Porque**: TikTok é rápido. Ritmo lento faz pessoas sair.
**Exceção**: Nenhuma.
**Aplicação**: Cortes rápidos, transições suaves, movimento constante.

### 4. Se não há movimento visual, ENTÃO menos retenção
**Porque**: TikTok favorece movimento (cortes, transições, efeitos).
**Exceção**: Nenhuma.
**Aplicação**: Sempre há mudança visual a cada 1-2 segundos.

### 5. Se som é fraco, ENTÃO menos alcance
**Porque**: TikTok prioriza conteúdo com som (trending audio).
**Exceção**: Nenhuma (som é crítico).
**Aplicação**: Use trending audio, não silence.

### 6. Se não usa trending audio, ENTÃO menos alcance
**Porque**: TikTok favorece conteúdo com trending audio.
**Exceção**: Nenhuma (trending audio é crítico).
**Aplicação**: Sempre use trending audio (check TikTok Discover).

### 7. Se engajamento é baixo, ENTÃO algoritmo não promove
**Porque**: TikTok usa engajamento como sinal de qualidade.
**Exceção**: Nenhuma.
**Aplicação**: Responda comentários, crie conteúdo que gera engajamento.

### 8. Se conteúdo é muito "polido", ENTÃO menos autêntico
**Porque**: TikTok favorece conteúdo autêntico, não muito produzido.
**Exceção**: Nenhuma.
**Aplicação**: Qualidade boa, mas não "Hollywood".

### 9. Se não há CTA claro, ENTÃO menos conversão
**Porque**: Sem CTA, pessoas não sabem o que fazer.
**Exceção**: Nenhuma.
**Aplicação**: "Siga para mais", "Clique no link", "Comente abaixo".

### 10. Se publica em horário errado, ENTÃO menos impressões
**Porque**: Horário afeta quantas pessoas veem.
**Exceção**: Nenhuma.
**Aplicação**: Melhor: 7-9am, 12-1pm, 6-9pm (horário local).

### 11. Se não há consistência, ENTÃO crescimento lento
**Porque**: Algoritmo favorece criadores consistentes.
**Exceção**: Nenhuma.
**Aplicação**: Publique 3-5 vezes por semana.

### 12. Se não há variedade de conteúdo, ENTÃO menos alcance
**Porque**: Algoritmo favorece diversidade.
**Exceção**: Nenhuma.
**Aplicação**: Varie entre educação, entretenimento, inspiração.

### 13. Se não há call-to-action para seguir, ENTÃO menos seguidores
**Porque**: Sem CTA, menos pessoas seguem.
**Exceção**: Nenhuma.
**Aplicação**: "Siga para mais dicas", "Siga para não perder".

### 14. Se vídeo é muito longo sem ponto de virada, ENTÃO pessoas saem
**Porque**: Sem cliffhanger, pessoa não continua.
**Exceção**: Nenhuma.
**Aplicação**: Cliffhanger a cada 15 segundos.

### 15. Se não há texto on-screen, ENTÃO menos compreensão
**Porque**: Texto on-screen ajuda compreensão (importante para som desligado).
**Exceção**: Nenhuma.
**Aplicação**: Sempre há texto on-screen que resume.

### 16. Se não há retargeting, ENTÃO deixas dinheiro na mesa
**Porque**: Pessoas que viram mas não clicaram podem clicar depois.
**Exceção**: Nenhuma.
**Aplicação**: Retargete com anúncios.

## Checklist Rápido de TikTok

- [ ] Hook nos primeiros 0.5s?
- [ ] Vídeo entre 15-60s?
- [ ] Ritmo rápido?
- [ ] Movimento visual a cada 1-2s?
- [ ] Som é trending audio?
- [ ] Engajamento esperado?
- [ ] Conteúdo autêntico?
- [ ] CTA claro?
- [ ] Publicado em horário certo?
- [ ] Consistência (3-5x/semana)?
- [ ] Variedade de conteúdo?
- [ ] CTA para seguir?
- [ ] Cliffhanger a cada 15s?
- [ ] Texto on-screen?
- [ ] Retargeting planejado?

## Estrutura de Vídeo TikTok

**0-0.5s**: Hook visual/verbal (curiosidade, emoção)
**0.5-5s**: Problema ou setup
**5-15s**: Solução ou desenvolvimento
**15-20s**: Cliffhanger ou ponto de virada
**20-60s**: Conclusão ou CTA
